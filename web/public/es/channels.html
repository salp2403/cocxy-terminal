Canales de actualización

Cocxy publica tres apps macOS firmadas con identificadores de bundle, feeds de actualización, preferencias y casks Homebrew separados. Puedes instalarlas juntas en /Applications.

Stable

Releases de producción para uso diario. Stable sigue tags públicos y el feed Sparkle principal.

brew tap salp2403/tap && brew install --cask cocxy

Feed: appcast.xml

Vista previa

Builds anticipadas firmadas para validar cambios antes del siguiente release stable.

brew tap salp2403/tap && brew install --cask cocxy-preview

Feed: appcast-preview.xml

Nightly

Builds diarias firmadas para probar el estado actual de desarrollo. Mantén stable para trabajo regular.

brew tap salp2403/tap && brew install --cask cocxy-nightly

Feed: appcast-nightly.xml

FAQ de canales

Puedo instalar todos los canales a la vez?
Sí. Stable usa dev.cocxy.terminal, vista previa usa dev.cocxy.terminal.preview y nightly usa dev.cocxy.terminal.nightly.

Puedo cambiar el feed dentro de la app?
Sí. Abre Preferencias y elige Stable, Vista previa o Nightly en Canal de actualización. El canal seleccionado cambia el feed Sparkle sin reinstalar la app.

Dónde quedan las preferencias?
Stable conserva ~/.config/cocxy/. Vista previa y nightly usan directorios por bundle para que probar builds no sobrescriba la configuración stable.
