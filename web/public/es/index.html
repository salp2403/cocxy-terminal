Hecha para desarrollo diario con agentes.

Cocxy combina terminal, revisión, documentación, remoto y automatización local en una app nativa.

Detección de agentes

Hooks, secuencias OSC, patrones y heurísticas locales detectan estado de agentes sin enviar salida de terminal a servidores.

Revisión integrada

Panel nativo para ver diffs, comentar líneas, aceptar o revertir hunks y devolver comentarios al agente desde el flujo local.

Espacio Markdown

Editor y previsualización con Mermaid, KaTeX, callouts, footnotes, QuickLook offline, búsqueda y exportación local.

Sesiones remotas

SSH, perfiles, Keychain opcional, túneles, SFTP y tmux para espacios remotos persistentes sin backend propietario.

Navegador integrado

Perfiles, DevTools, favoritos, descargas y previsualización de localhost en una división junto a la terminal.

Cero telemetría

Sin analytics, tracking ni reporte automático de crashes. La app funciona localmente y la red es explícita.

Renderizado GPU

CocxyCore usa Metal para scroll fluido, atlas de glifos, protocolos de imagen inline y diagnósticos de terminal.

CLI local

API por Unix socket para manejar pestañas, sesiones, agentes, remotos, dashboards, plugins y revisiones desde shell.

Sistema de plugins

Plugins por eventos con ejecución en sandbox para adaptar Cocxy al flujo de cada proyecto sin backend central.

Config por proyecto

Un archivo .cocxy.toml permite ajustar fuentes, opacidad, detección y atajos con recarga en caliente.

Terminal web local

Expone una terminal local por HTTP con frontend sin dependencias y control de frame rate, conexiones y attach.

Integración de shell

Integración nativa para zsh, bash y fish con CWD, límites de comando y bootstrap que respeta tu setup.

Preguntas rápidas.

¿Cocxy es gratis?

Sí. Cocxy es gratuito bajo licencia MIT. No requiere cuenta, suscripción ni login para funcionar.

¿Cocxy recopila datos?

No. Cocxy no incluye telemetría, analytics, tracking ni reporte automático de crashes.

¿Puedo usarla como terminal diaria?

Sí. Incluye pestañas, divisiones, búsqueda, integración de shell, SSH, CLI, temas, imagen inline y renderizado GPU.
