Cocxy Terminal

Tu terminal entiende flujos con agentes de código.

Terminal nativa para macOS con detección local de agentes, revisión de código integrada, workspace Markdown, sesiones remotas persistentes y cero telemetría. Construida en Swift. Licencia MIT.

Hecha para desarrollo diario con agentes.

Cocxy combina terminal, revisión, docs, remoto y automatización local en una app nativa.

Detección de agentes

Hooks, secuencias OSC, patrones y heurísticas locales detectan estado de agentes sin enviar salida de terminal a servidores.

Revisión integrada

Panel nativo para ver diffs, comentar líneas, aceptar o revertir hunks y devolver feedback al agente desde el flujo local.

Workspace Markdown

Editor y preview con Mermaid, KaTeX, callouts, footnotes, QuickLook offline, búsqueda y exportación local.

Sesiones remotas

SSH, perfiles, Keychain opcional, túneles, SFTP y tmux para workspaces remotos persistentes sin backend propietario.

Renderizado GPU

CocxyCore usa Metal para scroll fluido, atlas de glifos, protocolos de imagen inline y diagnósticos de terminal.

Cero telemetría

Sin analytics, tracking ni reporte automático de crashes. La app funciona localmente y la red es explícita.

Preguntas rápidas.

¿Cocxy es gratis?

Sí. Cocxy es gratuito bajo licencia MIT. No requiere cuenta, suscripción ni login para funcionar.

¿Cocxy recopila datos?

No. Cocxy no incluye telemetría, analytics, tracking ni reporte automático de crashes.

¿Puedo usarla como terminal diaria?

Sí. Incluye tabs, splits, búsqueda, shell integration, SSH, CLI, temas, imagen inline y renderizado GPU.
