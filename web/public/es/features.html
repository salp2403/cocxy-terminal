Funciones principales

Cocxy Terminal está diseñada como una app nativa para macOS: terminal diaria, paneles de desarrollo, automatización local y flujos con agentes sin backend propietario.

Detección local de agentes

Cocxy detecta sesiones de agentes de código mediante hooks, secuencias OSC, patrones y heurísticas de tiempo. Las señales se procesan localmente, por superficie, y alimentan dashboard, timeline, sidebar y rutas de feedback.

Panel de revisión de código

Muestra archivos modificados, diffs, comentarios inline, acciones por hunk y feedback estructurado hacia el agente. El objetivo es revisar cambios reales sin abandonar la terminal.

Workspace Markdown

Editor de fuente, preview local, Mermaid, KaTeX, callouts, footnotes, tabla de contenido, QuickLook offline, búsqueda, exportación HTML/PDF y helpers para imágenes.

Workspaces remotos

Perfiles SSH, ControlMaster, túneles, SFTP, tmux, daemon opcional y Keychain local para sostener sesiones remotas persistentes sin servidor Cocxy.

Plugins y CLI

Plugins con sandbox, eventos de terminal, permisos explícitos y timeout. La CLI cocxy controla tabs, splits, sesiones, agentes, hooks, Markdown, remoto, plugins y diagnósticos mediante socket local.

Cero telemetría

Sin analytics, tracking, reporte automático de crashes ni backend propietario. La red solo aparece cuando instalas, actualizas o usas una herramienta externa configurada por ti.
