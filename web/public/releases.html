Install via Homebrew

brew tap salp2403/tap && brew install --cask cocxy

Update: brew update && brew upgrade --cask cocxy

Auto-Updates

Cocxy uses Sparkle with EdDSA signature verification for auto-updates. Each DMG is signed with a private key that stays on the build machine; the app bundle ships with the matching public key, so only official releases are ever installed. The update feed is cocxy.dev/appcast.xml.

Nightly Channel

Nightly builds ship experimental features between stable releases. They install side by side with the stable version using a separate bundle ID and update feed. Download from the Releases page — look for builds tagged with -nightly.
