Release history and changelog for Cocxy Terminal.

v0.1.0

Latest March 2026

Initial Release

  • GPU Terminal — Metal-accelerated rendering via libghostty
  • Agent Detection — Passive 3-layer detection for Claude Code, Codex, Gemini CLI, Aider, Kiro, OpenCode
  • Remote Workspaces — SSH multiplexing, port tunnels, SFTP browser, key management
  • Built-in Browser — Profiles, DevTools, bookmarks, history, find-in-page
  • Smart CLI — 43 commands via Unix socket API
  • Auto-Updates — Sparkle with EdDSA signature verification
  • Zero Telemetry — No analytics, no crash reporting, no tracking
