Built for what comes next.

Every feature designed for developers who ship code with AI agents.

4-Layer Agent Detection

Hooks, OSC sequences, pattern matching, and timing heuristics. Detects Claude, Codex, Gemini, Aider, Copilot and Cursor in real time. Dashboard, timeline, and smart routing built in.

Remote Workspaces

SSH multiplexing with persistent sessions, SOCKS5 + HTTP CONNECT proxy with system-wide macOS integration, multi-channel agent relay with HMAC auth, and a POSIX remote daemon. Zero server installation.

Built-in Browser

Profiles, DevTools, bookmarks. Split browser alongside terminal. Preview localhost without leaving your workflow.

Zero Telemetry

No PostHog. No Sentry. No tracking of any kind. Your terminal, your data. Period.

GPU Terminal

Metal-accelerated rendering via libghostty. 120fps buttery smooth scrolling. Native performance, no Electron.

65 CLI Commands

Unix socket API for full scriptability. Manage tabs, agents, plugins, remote sessions, and dashboards from the command line.

Plugin System

Event-driven plugins with sandboxed execution. Respond to 8 terminal events with custom scripts. Extend Cocxy to fit your workflow.

Per-Project Config

Drop a .cocxy.toml in any project to override fonts, opacity, detection rules, and shortcuts. Hot-reload included. AppleScript automation built in.
